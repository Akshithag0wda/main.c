PROMPT 4:
--------------------------------------------------------------------------------
Write a complete C program for a 2D Terminal Graphics Editor.

Requirements:

1. Use a 20x60 character canvas initialized with '_'.
2. Support the following shapes:
   - Line
   - Rectangle
   - Circle
   - Triangle

3. Use:
   - Enum for shape types
   - Structures for each shape
   - Union inside a Shape structure
   - Global array to store up to 100 shapes

4. Implement:
   - Bresenham's Line Algorithm
   - Midpoint Circle Algorithm
   - Rectangle using four lines
   - Triangle using three lines

5. Features:
   - Display canvas with coordinate numbers
   - Add objects through a menu
   - Assign a unique ID to each object
   - Delete objects using their ID
   - List active objects
   - Clear all objects
   - Exit program

6. Input validation:
   - Use fgets() and strtol() for reading integers
   - Reject invalid input and out-of-range values

7. Menu:

   1. Display Canvas
   2. Add Object
   3. Delete Object
   4. Clear All Objects
   5. Exit

8. Follow modular programming:
   - clear_canvas()
   - display_canvas()
   - set_pixel()
   - draw_line()
   - draw_rectangle()
   - draw_circle()
   - draw_triangle()
   - render_shapes()
   - add_shape_menu()
   - delete_shape_menu()

9. Use only standard C libraries:
   stdio.h, stdlib.h

10. Generate the complete compilable code with comments explaining each function.

--------------------------------------------------------------------------------
ANSWER 4:
--------------------------------------------------------------------------------
Writes the final source implementation of the editor to `main.c` satisfying all 10 specifications:
- Canvas grid size expanded to 20x60.
- Shape indexing replaced with unique ID assignment and lookup deletion.
- Integers read through `fgets()` and validated with `strtol()`.
- Rectangle routine modified to call `draw_line()` exactly four times.
- Main menu simplified to 5 distinct options.
- `<math.h>` references removed, relying exclusively on `<stdio.h>` and `<stdlib.h>`.
- Code modularized exactly under the required function names:
  `clear_canvas()`, `display_canvas()`, `set_pixel()`, `draw_line()`, `draw_rectangle()`, `draw_circle()`, `draw_triangle()`, `render_shapes()`, `add_shape_menu()`, and `delete_shape_menu()`.
